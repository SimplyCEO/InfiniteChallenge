20w14infinite InfiniteChallenge
===============================

This Data Pack provides a challenge for `20w14infinite` game version.

Description
-----------

The main goal is to reach the The End dimension,
but it is all up to the player what the real goal is.

The player will spawn inside a chamber with a small amount of portals.

Each portal is a one way dimension.
After some seconds the portal will be deactivated.
It gives time for the player to choose the home dimension since some can cause game crashes and/or photosensitivity triggers.

The player can return to the portal hub whenever going to the reset dimension: The Nether.

Installation
------------

Download the repository and move the folder `data_pack` to the Minecraft world folder.

A better way to access the Minecraft world folder
is to click on <b>Edit</b>, after selecting the desired world, and <b>Open World Folder</b>.

Put inside `datapacks`.

Optional: Rename it as `InfiniteChallenge`.
